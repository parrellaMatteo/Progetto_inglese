What is Cloud Gaming?

It is like "Netflix for games." The game is running on a high-end PC in a remote data center, and you receive a video feed of the action in real-time.

The Enemy: Latency

This is the most technical challenge. Delay is called Latency or Lag. For a 10/10 experience, you need a latency below 50ms.

Requirements

Stable 5GHz Wi-Fi or Ethernet is mandatory. It consumes up to 10GB per hour, much more than a standard Netflix movie!
